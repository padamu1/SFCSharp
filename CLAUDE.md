# Claude Guidelines

## 프로젝트 개요

- 이 레포지토리는 Unity에서 사용자가 코드 및 프리팹을 만들고 이를 업로드하면 다른 유저들이 다운받아 사용할 수 있게 해주는 모드 툴 역할의 라이브러리이다.

## 작업 완료 후 검사

- 모든 작업이 끝나면 작업한 내용을 검사한다. 불필요한 코드나 정리가 필요한 코드가 있으면 정리한다.

## 작업 기본 방향

- 작업의 기본 방향은 사용자가 C# + Unity를 사용하는 환경과 다름이 없게 한다.

## IL2CPP 모바일 호환성

- 이 라이브러리는 IL2CPP 빌드 환경(모바일 포함)을 지원해야 한다.
- System.Reflection.Emit, 런타임 코드 생성, 동적 어셈블리 로딩을 사용하지 않는다.
- RegexOptions.Compiled를 사용하지 않는다 (AOT 환경에서 문제 발생).
- 제네릭 가상 메서드의 값 타입 조합에 주의한다 (AOT에서 코드 생성 불가).
